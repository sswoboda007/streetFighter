# Street Fighter

## Problem
a competitive platform for players, a way to connect gamers, a space for friendly matches

## Users
- casual gamers looking for fun
- competitive players wanting tournaments
- friends wanting to challenge each other

## Features
- create character profiles
- have ranked matches

## Tech Stack
- **language**: Python
- **framework**: Flask

## File Tree
- `app.py` - Main Flask application file that initializes the app and sets up routes.
- `requirements.txt` - File listing the required Python packages for the project.
- `templates/index.html` - HTML template for the main page where players can view character profiles and match options.
- `static/styles.css` - CSS file for styling the web application's interface.
- `static/scripts.js` - JavaScript file for handling client-side interactions and dynamic content.
- `models/character.py` - Python class definition for character profiles, including attributes and methods.
- `tournaments.py` - Python module to manage tournament scheduling and matchmaking.
- `README.md` - Documentation file that provides an overview of the project, setup instructions, and feature details.
