# Street Fighter Project

## Overview
Street Fighter is a competitive platform for players to connect, play, and engage in friendly matches. It offers features tailored for both casual gamers and competitive players.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/street-fighter.git
   ```
2. Navigate to the project directory:
   ```
   cd street-fighter
   ```
3. Set up a virtual environment:
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
4. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

5. Run the application:
   ```
   python app.py
   ```
